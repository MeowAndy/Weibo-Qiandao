# 微博超话签到 Web

这是 Weibo-Qiandao 的网页端源码目录。完整使用说明见项目根目录 `README.md`。

## 快速启动

推荐双击项目根目录：

```text
一键启动网页端.bat
```

也可以手动启动：

```powershell
npm install
npm start
```

访问：

```text
http://localhost:3000
```

## 重要文件

- `server.js`：后端服务。
- `public/`：前端页面。
- `config.example.json`：配置模板。
- `config.json`：本地配置，默认不提交。
- `data/`：账号、Cookie、日志数据，默认不提交。
